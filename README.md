# Инструкция по установке программ

1) Скачать архив необходимых для работы прогармм по ссылке: https://github.com/cldcdr42/ApplicationCapybara/releases/download/v1.0.5/DataAcqSoft.zip
2) Распаковать содержимое папки DataAcqSoft.zip на рабочий стол
3) Выполнить установку необходимых драйверов и программ:  
   3.1) Перейти в папку "0 - Драйверы и необходимые компоненты" внутри распакованной папки DataAcqSoft  
   3.2) Запустить программу "CH341SER.EXE", нажать на кнопку INSTALL, дождаться окончания установки  
   3.3) Запустить "NeoRec-1.5.13646.exe" и следовать шагам по установке. Можно пропустить этот шаг если данная прогамма уже установлена на компьютере  
   3.4) Перейти в папку "MCSUSBDriver_32_64_bit_auto" и запустить "setup.exe", следовать шагам по установке  

# Подключение миографа к компьютеру

1) Подключить миограф по USB к компьютеру  
2) Запустить программу NeoRec  
3) Нажать на кнопку Профиль и перейти во вкладку Основные настройки  
  3.0) Создать новый профиль нажатием на зеленый плюс  
	3.1) Выставить схему аксессуаров - standard: NVX.NVX36.NVX36CompatibleCap  
	3.2) Выставить формат записи - Lab Streaming Layer (LSL)  
	3.4) Указать частоту дискретизации - 1000 Гц  
	3.5) Указать режим работы - Стандартный  
	3.6) Указать латентность LSL - Очень быстрый  
4) Перейти на вкладку Каналы  
  4.0) Добавить нвую группу каналов  
	4.1) Добавить новый канал  
	4.2) Указать тип канала - ЭМГ  
	4.3) Указать источник канала - B1+B1-  
	4.4) Указать усиление (sensor gain) - 1  
	4.5) Указать смещение (sensor offset) - 0  
	4.6) Указать фильтры  
        4.6.1) ФВЧ - 10 Гц  
        4.6.2) ФНЧ - Выкл  
        4.6.3) РФ - 50 Гц (отфильтровать ток)  
5) Нажать ОК, вернуться на основной экран программы  
6) Запустить сбор данных (кнопка Старт) и трансляцию данных (кнопка Запись)  

# Подключение ортеза к компьютеру

1) Для подключения ортеза к компьютеру необходим провод USB Type C, подключить его к компьютеру (разъем Type C вставляется в слот, расположенный в корпусе ортеза. Разъем USB - в USB слот компьютера)

# Инструкция по работе с программой сбора данных

0) Подключить ортез к компьютеру, подключить миограф и начать трансляцию
1) Перейти в папку "1 - Программа сбора данных" и запустить файл "main.exe"
2) В открывшемся окне нажать START для начала сбора данных. Собираемые данные в реальном времени выводтся на экран
3) После окончания сеанса записи нажать STOP
4) Нажать открыть расположение файла - откроется папка, которая содержит все записанные сеансы. Название папки сеанса состоит из даты и времени его проведения
5) Открыть папку нужного сеасна
6) Использовать файлы emg.csv (данные с миографа), mcu.csv (данные с ортеза) и session_info.txt (данные о сессии, вводимые пользователем в правой части окна программы) для анализа полученных результатов сторониими программами
